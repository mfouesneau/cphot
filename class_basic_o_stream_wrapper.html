Detailed Description

template<typename StreamType>
class BasicOStreamWrapper< StreamType >

Wrapper of std::basic_ostream into RapidJSON's Stream concept.

The classes can be wrapped including but not limited to:

  • std::ostringstream
  • std::stringstream
  • std::wpstringstream
  • std::wstringstream
  • std::ifstream
  • std::fstream
  • std::wofstream
  • std::wfstream
